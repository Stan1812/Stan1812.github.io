CSS实现有趣的小部件

Veröffentlicht am 2017-03-25 | in 技术

CSS实现转动圆环欢迎界面

这应该全是一个比较简单的小组件，但是个人感觉比较有意思，所以现在来写篇文。

组件的基本HTML文档如下:

&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
    &lt;title&gt;圆形动画&lt;/title&gt;
    &lt;link rel="stylesheet" type="text/css" href="4.css"&gt;
&lt;/head&gt;
&lt;body&gt;
&lt;div class="demo"&gt;
    &lt;div class="pic"&gt; &lt;img src="53cdcf5edc0dd.jpg"&gt;&lt;/div&gt;
    &lt;div class="mask"&gt;
        &lt;h2&gt;WELCOME&lt;h2&gt;
    &lt;/div&gt;
    &lt;div class="border"&gt;&lt;/div&gt;
&lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;`</pre>
&nbsp;

下面来实现具体的效果：

1：鼠标划过时边框转动

2：鼠标划过主体加上一层蒙版，标题放大

&nbsp;

首先将组件的基本结构写一下：
<pre class="pure-highlightjs">`body{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
｝
.demo, .pic, .mask, .border{
    width:     300px;
    height: 300px;
    position: absolute;
    margin-left: 50%;
    left: -150px;
    border-radius:     50%;
}
.demo{
    top: 150px;
}
.pic img{
    width:     300px;
    height: 300px;
    position: absolute;
    border-radius:     50%;    
}

.mask h2{
    display: table;
    position: absolute;
    margin-left: 50%;
    left: 50%;
    transform: translateX(-50%);
    top: 100px;
    color: #fff;
    margin:0 auto;
    line-height: 100px;
}
.border{
    border:10px solid ;
    border-top-color: #3ba8dc;
    border-left-color:#5f8da3;
    border-right-color: #3abcb5;
    border-bottom-color: #a6ebe7;
    transform: rotate(0deg);
    box-sizing: border-box
}

`</pre>
&nbsp;

我们在这里使用border-redius：50％来实现圆形边框。在这里复习一下利用border来画一些简单图形的方法：https://gold.xitu.io/entry/58cf8cbd0ce4630057d96f76

&nbsp;

接下来就是简单的动画效果咯：
<pre class="pure-highlightjs">`a{
    text-decoration: none;}
.demo:hover .mask{
    background: rgba(0,0,0,0.3);
    transition:  all 0.3s;
}
.demo:hover .mask  h2{
    font-size: 150%;
    font-weight: 300;
    transition:  all 0.3s;
}
.demo:hover .border{
    transform: rotate(1080deg);
    transition: all 2s ease-in;
}

好，这个很简单的东西就完了～

CSS3实现文字流光效果

Veröffentlicht am 2017-03-25 | in 技术
在百度前端技术学院（这里）上看到一个比较有意思的效果，@陈慕杰照着实现了一遍，现在来分享一下，最终要实现的效果：

实现文字的流光渐变动画
背景图需要进行模糊处理
实现按钮边框的从中间到两边扩展开
下面来一步步实现这些效果。

页面结构：



<!DOCTYPE html>
<html lang=”en”>

<div class=”box-wrapper”>

<h1 class=”title”>welcome to our world</h1>
</div>
</div>

</div>
</body>
</html></pre> 基本样式： <pre class="pure-highlightjs"> {

.box{
position: relative;

height:100%;
padding-top: 80px;
}</pre> 现在我们已经实现了页面的基本布局和样式，container容器固定宽高，在页面中水平居中，包含box-wrapper和img两个盒子，box-wrapper充满container，绝对定位使文字悬浮与图片上面。下面来实现文字的流光效果。 流光效果的主要原理：将文字的背景色的宽度拉长至原来的两倍，而在background-image中设置两份相同的颜色组，将背景拉长之后便只显示一份颜色组，超出背景宽度的颜色组通过动画改变背景色的位置background-position来实现流光效果。 &nbsp; <pre class="pure-highlightjs">.title {
background-image: -webkit-linear-gradient(left,#D81159, #E53A40 10%, #FFBC42 20%, #75D701 30%, #30A9DE 40%,#D81159 50%, #E53A40 60%, #FFBC42 70%, #75D701 80%, #30A9DE 90%,#D81159);
-webkit-background-clip: text;/来使文字作为裁剪区域向外裁剪，此时文字的颜色将覆盖在渐变色之上。/
color:transparent;/让文字显示渐变色，将文字颜色变为透明/
transform: translate(0,20px);/实现鼠标滑过的上浮效果/
background-size: 200% 100%;
animation: flowlight 5s linear infinite;
}

@keyframes flowlight {
0% {
background-position: 0 0;
}
100% {
background-position: -100% 0;
}
}
.container:hover .title{
transform: translate(0);
}
</pre> 现在我们已经实现了流光动画的效果 接下来实现按钮边框的从中间到两边扩展开，这个效果主要使用了伪元素::before、::after分别来实现容器的左右和上下边框，伪元素边框使用绝对定位，父容器box使用相对定位，伪元素边框位置分别定位到相对父元素的top和left的50%，通过改变其位置和宽高来实现边框从中间到两边扩展的动画效果。 <pre class="pure-highlightjs">.box::before {

}

.box::after {
content:’’;
border:3px solid #fff;
border-width: 3px 0;
position: absolute;
width:0;
height:100%;
top:0;
left:50%;
box-sizing: border-box;
}
/
实现边框扩展的的动画效果，并加上过渡效果。 */

.container:hover .box::before{
height:100%;
top:0;

.container:hover .box::after {
width: 100%;

.box::before {

transition: all .8s;
}
</pre> </div> &nbsp; 背景图的模糊处理 从最终实现我们可以看到，鼠标悬浮时背景图会出现模糊效果，这样的模糊效果我们可以通过CSS3的滤镜属性filter来实现。 filter：CSS滤镜属性，可以在元素呈现之前，为元素的渲染提供一些效果，如模糊、颜色转移之类的。滤镜常用于调整图像、背景、边框的渲染。 在这里我们需要设置filter的blur来实现对象的模糊效果。 <pre class="pure-highlightjs">.container:hover .img {

-webkit-filter: blur(2px);
-moz-filter: blur(2px);
-ms-filter: blur(2px);
}
</pre> 加上之后图片把文字和边框覆盖了，这里我们需要设置box-wrapper容器的z-index属性，来凸显各个容器的层级。 <pre class="pure-highlightjs">.box-wrapper {

}</pre> 这样，背景模糊的效果就实现了。 最后，我们的显示隐藏效果还没实现。这里，我们通过设置透明度opacity来实现显示隐藏，设置transtion来实现过渡效果。 <pre class="pure-highlightjs"> .box{ opacity: 0; transition: opacity .8s; }
.title { transition: transform .8s,opacity .8s; }
.container:hover .box{ opacity: 1; }
.container:hover .title{ opacity: 1; }

现在，我们已经实现了所有的动画效果，总结：

Veröffentlicht am 2017-03-24 | in 感悟

前端小学生的幼儿园生涯感悟

从正式开始学习前端大概有一个多月了，前端确实有着比较门槛比较低的特点，不过想要做好还是并不容易。现在应该说算是踏入了前端的缤纷世界了吧，虽然技术还很垃圾但是也能写出一点东西。今天有空，那就写一点东西吧。

学习前端是从《Head First HTML与CSS》以及学长分享的网易前端工程师课程视频开始的/在此推荐下那本书，很不错。刚开始就是按着书上的代码敲，刚开始是直接把css样式嵌入到html文件中的，后来跟着书慢慢的把样式表放到一个独立的文件中，书中并没有一下子把你需要的知道的东西都告诉你，而是从不完善到完善的过程，这也让我知道了很多东西为什么要那样做而不是这样做。敲出的第一个demo是基本上按照书上的形式来的，非常简单的一个小demo，不过是应用了一些居中，边框，背景色什么很简单的东西。不怕被笑话这里就是第一个小网页https://github.com/Stan1812/first-github

学了一点之后还是继续按照书上的方式来敲demo，不算平时的练习，这应该是我第二个比较正式的有头有脚的页面，这个页面的布局是使用的表格式布局/因为当时还不熟悉div+css的布局方式/这个页面是三列的那种比较典型的网页，现在看来真是内容简单布局死板不利于以后的改动，给自己留的坑。

当这本书看完之后我已经将这个小页面加了一个文章页和一个表单/虽然并没有后台/

之后看的书就比较杂了比较经典的是《精通CSS 高级web标准解决方案》，在微信公众号，知乎和csdn， QQ微信群等乱七八糟各种渠道看其他人写的各种博客/在此推荐SCDN 掘金这两个社区，很6/真的是走路都在看，感觉很有意思。平时也看了很多慕课上面的课程，然后平时也有很多乱七八糟的小东西的练习。写比较完整的是这个网页，是按照慕课上面一个扁平风格博客的教程看的，然后按照他的样式自己实现了一遍。demo在这里。

前端除了HTML和CSS自然也少不了JS这个老大哥，刚开始接触JS的时候觉得面向对象编程是什么鬼/虽然有C的基础，但是我们学校变态把C和C++放在一起讲了，然而C++的面向对象就只是简单的介绍了一下。。。估计下次在大学课堂接触面向对象编程应该是在大二下或大三学java的时候了/写JS的时候真的是一脸懵逼啊。。。虽然现在依然是懵逼，但可能剩下了半脸懵逼了吧。。。俗话说得好，不能写脚本的程序员不是程序员，不能写好脚本的程序员不是好程序员。嗯，虽然html严格的说并不是程序语言而是标记语言。。JS还是要继续努力学啊！骚年！

腾讯云搭建个人博客

Veröffentlicht am 2017-03-23 | in 技术

腾讯云服务器wp建站

（windows sever+xampp+wordpress）

都说搞技术的人都要有一个自己的博客，便想着折腾折腾，于是就搭了个个人博客。在腾讯云买了个服务器，学生机，只要通过认证就只需要一个月一块钱，当然性能是相对较弱了，（在此不当性能党）不过一个博客是完全够用了。而对于没玩过服务器的我来说真的是一点都不好搞，下面就写一篇简单的教程看来给大家分享一下。

本来是准备浪一波来自己配置环境的，后来意识到这对于没有配置过这种环境的人来说并太友好，折腾了好久也没有搞好，后来在网上找教程发现了这个建站神器XAMPP。也给以后的同志们提供一点建议。

XAMPP是一个功能强大的建站集成软件包，结合学生机Windows Server 2012 R2搭建服务器端。我使用的是Windows Server 2012 R2公共镜像（当然据说在服务器端还是centos比较好，但是我这个小学生还没有学过linux，所以就选择了熟悉的windows）

第一步当然就是登陆到你的服务器了，可以在云控制台登陆，也可使用win自带的运程连接工具来连接win+R运行mstsc进入界面，输入ip ，确定，弹出另一界面，输入密码（这个密码可以很方便的改的，忘记关系也不大）中间会有提示不信任这个ip，不用管，继续登陆即可。然后就顺利的登陆你的服务器端了。

首先会弹出服务管理器，在我们这种方法中不需要自己去设置，关闭就行，下载XAMPP与wordpress到本地或直接在服务器端下载，（在服务器端下载速度还是刻意接受的，不过在此推荐下到本地，然后上传到服务器（依然是使用远程连接：显示选项-本地资源））最终目的是把压缩包搞到服务器里面。 第二步就要开始安装XAMPP了，一直点next（注意安装路径一定不能有特殊字符，不然很容易出现错误）安装成功后就开始启动按住安装Apache和MySQL服务了，（点击那个‘×’的图标安装Apache和MySQL服务（安装好后变为‘√’），并分别点击”Start”开始，当图标变绿时表明成功开启服务）

而在启动这两项服务时也会遇到启动不成功的现象，在网上有很多教程这里就不再重复了，基本上是端口的问题。如果你不想搞，最简单的方法也不过就是重装一下系统嘛，还是很快的。这里分享一下csdn的一篇文点我

接下来进行数据库的创建，点击MySQL中的Admin进入phpMyAdmin(一款强大的MySQL数据库管理软件)，用户名和密码默认均为root 进入数据库后新建一个数据库（在这里我创建的是wordpress）用来存放wordpress中的数据 接下来就是wordpress解压安装了，将解压后wordpress中的所有文件放入xampps\htdocs之中（原先存在的几个文件，对，都给删喽~） 浏览器中输入127.0.0.1，点击创建配置文件。

点击提交就成功啦，进入wordpress后台管理界面，跟微信公众号后台性质一样，这样就创建了独属于你自己的个人网站。 然后就可以通过域名ip地址访问了，还有一步无所谓先后的就是把你买的域名解析到这个是地址就好了，我用的是阿里云的解析（因为我在阿里买的top的顶级域名）。这一切都完成之后就可以愉快的在各个平台登陆你的博客了，如我的域名：www.ilibiao.top。 然而并没有完，当你设置好wordpress的主题之后，你正以为要大功告成的时候，你会发现在浏览器打开的时候网站没有CSS样式，只有HTML文件，这时只需要在wp后台-设置-常规-WordPress地址U 点地址（URL）改为你的ip即可，之后再在浏览器打开就能愉快的装逼访问了。

在搭建的过程中肯定会遇到不少的问题，虽然一下子就装好很爽，但慢慢自己配置，然后找到问题，解决问题的快感也是无法比拟的。 这也正是乐趣所在啊~
