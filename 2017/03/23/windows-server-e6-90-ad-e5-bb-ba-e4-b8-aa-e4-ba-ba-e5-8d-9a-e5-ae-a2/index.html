    腾讯云服务器wp建站

    （windows sever+xampp+wordpress）

    都说搞技术的人都要有一个自己的博客，便想着折腾折腾，于是就搭了个个人博客。在腾讯云买了个服务器，学生机，只要通过认证就只需要一个月一块钱，当然性能是相对较弱了，（在此不当性能党）不过一个博客是完全够用了。而对于没玩过服务器的我来说真的是一点都不好搞，下面就写一篇简单的教程看来给大家分享一下。

    本来是准备浪一波来自己配置环境的，后来意识到这对于没有配置过这种环境的人来说并太友好，折腾了好久也没有搞好，后来在网上找教程发现了这个建站神器XAMPP。也给以后的同志们提供一点建议。

    XAMPP是一个功能强大的建站集成软件包，结合学生机Windows Server 2012 R2搭建服务器端。我使用的是Windows Server 2012 R2公共镜像（当然据说在服务器端还是centos比较好，但是我这个小学生还没有学过linux，所以就选择了熟悉的windows）

    第一步当然就是登陆到你的服务器了，可以在云控制台登陆，也可使用win自带的运程连接工具来连接win+R运行mstsc进入界面，输入ip ，确定，弹出另一界面，输入密码（这个密码可以很方便的改的，忘记关系也不大）中间会有提示不信任这个ip，不用管，继续登陆即可。然后就顺利的登陆你的服务器端了。

    首先会弹出服务管理器，在我们这种方法中不需要自己去设置，关闭就行，下载XAMPP与wordpress到本地或直接在服务器端下载，（在服务器端下载速度还是刻意接受的，不过在此推荐下到本地，然后上传到服务器（依然是使用远程连接：显示选项-本地资源））最终目的是把压缩包搞到服务器里面。 第二步就要开始安装XAMPP了，一直点next（注意安装路径一定不能有特殊字符，不然很容易出现错误）安装成功后就开始启动按住安装Apache和MySQL服务了，（点击那个‘×’的图标安装Apache和MySQL服务（安装好后变为‘√’），并分别点击”Start”开始，当图标变绿时表明成功开启服务）

    而在启动这两项服务时也会遇到启动不成功的现象，在网上有很多教程这里就不再重复了，基本上是端口的问题。如果你不想搞，最简单的方法也不过就是重装一下系统嘛，还是很快的。这里分享一下csdn的一篇文点我

    接下来进行数据库的创建，点击MySQL中的Admin进入phpMyAdmin(一款强大的MySQL数据库管理软件)，用户名和密码默认均为root 进入数据库后新建一个数据库（在这里我创建的是wordpress）用来存放wordpress中的数据 接下来就是wordpress解压安装了，将解压后wordpress中的所有文件放入xampps\htdocs之中（原先存在的几个文件，对，都给删喽~） 浏览器中输入127.0.0.1，点击创建配置文件。

    点击提交就成功啦，进入wordpress后台管理界面，跟微信公众号后台性质一样，这样就创建了独属于你自己的个人网站。 然后就可以通过域名ip地址访问了，还有一步无所谓先后的就是把你买的域名解析到这个是地址就好了，我用的是阿里云的解析（因为我在阿里买的top的顶级域名）。这一切都完成之后就可以愉快的在各个平台登陆你的博客了，如我的域名：www.ilibiao.top。 然而并没有完，当你设置好wordpress的主题之后，你正以为要大功告成的时候，你会发现在浏览器打开的时候网站没有CSS样式，只有HTML文件，这时只需要在wp后台-设置-常规-WordPress地址U 点地址（URL）改为你的ip即可，之后再在浏览器打开就能愉快的装逼访问了。

    在搭建的过程中肯定会遇到不少的问题，虽然一下子就装好很爽，但慢慢自己配置，然后找到问题，解决问题的快感也是无法比拟的。 这也正是乐趣所在啊~
